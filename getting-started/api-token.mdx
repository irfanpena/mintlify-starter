---
title: 'Get an API Token'
description: 'This guide walks you through creating your MrScraper API Token.'
---

An API token allows your applications to securely access the MrScraper API. It acts as a unique identifier for authentication, ensuring that only authorized users can make requests.

Follow the steps below to generate one:

1. Click your **User Profile** on top right of the page, and select **API Tokens**.

    <Frame>
        ![](/images/api-token/step-1.png)
    </Frame>

2. Click **New Token**.

    <Frame>
        ![](/images/api-token/step-2.png)
    </Frame>

3. Enter a **name** and set an **expiration date** for your API token.

    <Frame>
        ![](/images/api-token/step-3.png)
    </Frame>

4. Click **Create**.
5. Copy your new API token and store it in a secure location.

    <Frame>
        ![](/images/api-token/step-4.png)
    </Frame>

<Warning>
    **Remember that your API token is a secret!**

    Never expose it in client-side code, such as browsers or apps. Always load your API token securely from environment variables or a server-side configuration.
</Warning>
