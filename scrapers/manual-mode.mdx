    <Step title="Configure your scraper by adding steps">
        You can add multiple steps to your scraper by clicking on the "Add Step" button. Each step allows you to define specific actions such as extract, click, delay, etc.

        <Frame>
            ![](/images/manual-mode-scraper/add-step.png)
        </Frame>

        ### Extract

        The Extract step allows you to specify the data you want to scrape from the webpage. You can set `Extraction Name`, select `Extraction Type` to `Text` (ex: Hello World), `Inner HTML` (ex: \<span\>hello world\<\/span\>), `Outer HTML` (ex: \<h1 class="bold">\<span>hello world\<\/span>\<\/h1>), and `Attribute` (ex: "class": bold), and define CSS selectors to target specific elements.

        <Frame>
            ![](/images/manual-mode-scraper/extract.png)
        </Frame>

        ### Click

        The Click step allows you to simulate a click action on a specified element.

        <Frame>
            ![](/images/manual-mode-scraper/click.png)
        </Frame>

        ### Delay

        The Delay step allows you to pause the scraper for a specified amount of time (in milliseconds) before proceeding to the next step.

        <Frame>
            ![](/images/manual-mode-scraper/delay.png)
        </Frame>

        ### input

        The Input step allows you to enter text into input fields on the webpage.

        <Frame>
            ![](/images/manual-mode-scraper/input.png)
        </Frame>

        ### Scroll

        The Scroll step allows you to scroll until the end, scroll until find a text scroll, or scroll to a specific element (or until the number of a specific element) on the webpage.

        <Frame>
            ![](/images/manual-mode-scraper/scroll.png)
        </Frame>

        ### Inject Javascript

        The Inject Javascript step allows you to run custom JavaScript code on the webpage.

        <Frame>
            ![](/images/manual-mode-scraper/inject.png)
        </Frame>

        ### Paginate

        Set up pagination to navigate through multiple pages automatically.

        | Pagination Type | Required Parameter |
        |-----------------|-------------|
        | Query pagination | Query Parameter (eg. "?page=2") |
        | Directory pagination (eg. "/list/page/2") | - |
        | Next page link | Next page selector |

        <Frame>
            ![](/images/manual-mode-scraper/paginate.png)
        </Frame>
    </Step>
